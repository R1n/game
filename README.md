# Getting Started
1.	Installation process:
```
npm i
```

2. create .env file with env from "example.env"

3. open new terminal and create new db:
```
psql -d postgres
createdb ancientgg
```

4. insert fake data in db  
  ``` 
npm run unpacking // (only for local setup)
```  

5. start
```
npm start
```
